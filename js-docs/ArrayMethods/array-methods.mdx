# Array Methods

**Array methods** are built-in functions that allow manipulation, iteration, and transformation of arrays.

## Creating Arrays

```javascript
const arr1 = [1, 2, 3];
const arr2 = new Array(1, 2, 3);
const arr3 = Array.from("hello"); // ["h", "e", "l", "l", "o"]
const arr4 = Array.of(1, 2, 3);
```

## Mutating Methods

These methods **modify** the original array:

| Method | Description |
|--------|-------------|
| `push()` | Add to end |
| `pop()` | Remove from end |
| `unshift()` | Add to beginning |
| `shift()` | Remove from beginning |
| `splice()` | Add/remove elements |
| `sort()` | Sort elements |
| `reverse()` | Reverse order |
| `fill()` | Fill with value |

```javascript
const arr = [1, 2, 3];
arr.push(4);      // [1, 2, 3, 4]
arr.pop();        // [1, 2, 3]
arr.unshift(0);   // [0, 1, 2, 3]
arr.shift();      // [1, 2, 3]
arr.splice(1, 1); // [1, 3] (remove 1 at index 1)
```

## Non-Mutating Methods

These return a **new array**:

| Method | Description |
|--------|-------------|
| `slice()` | Extract portion |
| `concat()` | Combine arrays |
| `map()` | Transform elements |
| `filter()` | Filter elements |
| `flat()` | Flatten nested arrays |

```javascript
const arr = [1, 2, 3];
arr.slice(1);        // [2, 3]
arr.concat([4, 5]);  // [1, 2, 3, 4, 5]
arr.map(x => x * 2); // [2, 4, 6]
```

## Search Methods

```javascript
const arr = [1, 2, 3, 4, 5];

arr.find(x => x > 3);      // 4 (first match)
arr.findIndex(x => x > 3); // 3 (index)
arr.includes(3);           // true
arr.indexOf(3);            // 2
arr.some(x => x > 4);      // true (any match)
arr.every(x => x > 0);     // true (all match)
```

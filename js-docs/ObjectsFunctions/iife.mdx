# IIFE

An **IIFE** (Immediately Invoked Function Expression) is a function that is defined and executed immediately after creation.

## Syntax

```javascript
(function() {
  // code here runs immediately
})();

// Arrow function IIFE
(() => {
  console.log("Arrow IIFE");
})();
```

## Why Use IIFE

- Create private scope
- Avoid global namespace pollution
- Execute initialization code
- Module pattern (pre-ES6)

```javascript
// Basic IIFE
(function() {
  let privateVar = "I'm private";
  console.log("IIFE executed!");
})();
// privateVar is not accessible here

// IIFE with parameters
(function(name, age) {
  console.log(`${name} is ${age} years old`);
})("John", 30);

// IIFE returning a value
const result = (function() {
  return 5 + 5;
})();
console.log(result); // 10
```

## Module Pattern

```javascript
const Calculator = (function() {
  let result = 0; // Private
  
  return {
    add(n) { result += n; return this; },
    subtract(n) { result -= n; return this; },
    getResult() { return result; },
    reset() { result = 0; return this; }
  };
})();

Calculator.add(5).add(3).subtract(2).getResult(); // 6
```

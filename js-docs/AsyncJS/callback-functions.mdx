# Async Callback Functions

In async context, callbacks execute when an operation completes.

```javascript
function fetchData(callback) {
  console.log("Fetching data...");
  setTimeout(() => {
    const data = { id: 1, name: "Product" };
    callback(data);
  }, 2000);
}

console.log("Before fetch");
fetchData((data) => {
  console.log("Data received:", data);
});
console.log("After fetch (doesn't wait)");

// Output:
// Before fetch
// Fetching data...
// After fetch (doesn't wait)
// (2 seconds later) Data received: {...}
```

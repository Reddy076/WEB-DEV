# Synchronous vs Asynchronous

**Synchronous**: Code executes line by line; each line waits for the previous to complete.

**Asynchronous**: Code can start an operation and move on; results are handled later.

```javascript
// Synchronous (blocking)
console.log("1");
console.log("2");
console.log("3");
// Output: 1, 2, 3

// Asynchronous (non-blocking)
console.log("1");
setTimeout(() => console.log("2"), 0);
console.log("3");
// Output: 1, 3, 2
```

## Why Async Matters

- JavaScript is single-threaded
- Without async, slow operations would freeze the page
- Enables non-blocking I/O operations

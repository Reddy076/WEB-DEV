# Selecting DOM Elements

Methods to access and reference HTML elements in JavaScript.

## Selection Methods

| Method | Returns | Description |
|--------|---------|-------------|
| `getElementById()` | Single element | Selects by ID |
| `querySelector()` | First match | CSS selector |
| `querySelectorAll()` | NodeList | All matches |
| `getElementsByClassName()` | HTMLCollection | By class |
| `getElementsByTagName()` | HTMLCollection | By tag |

```javascript
// By ID (fastest)
const header = document.getElementById("header");

// CSS selector (most flexible)
const button = document.querySelector("button");
const container = document.querySelector(".container");
const submit = document.querySelector("#form button[type='submit']");

// All matching elements
const allButtons = document.querySelectorAll("button");

// NodeList forEach
allButtons.forEach(btn => btn.classList.add("styled"));

// Convert to array
const buttonsArray = Array.from(allButtons);
```

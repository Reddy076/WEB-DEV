# Modifying DOM Content

Methods and properties to change the content, attributes, and styles of DOM elements.

## Content Modification

```javascript
const element = document.querySelector("#myElement");

// Text content
element.textContent = "Plain text only";      // Safest
element.innerText = "Visible text";           // Respects CSS
element.innerHTML = "<strong>Bold</strong>";  // Parses HTML (security risk!)
```

## Attribute Modification

```javascript
element.setAttribute("data-id", "123");
element.getAttribute("data-id");      // "123"
element.removeAttribute("disabled");
element.hasAttribute("disabled");     // false

// Direct property access
element.id = "newId";
element.className = "class1 class2";
```

## classList API

```javascript
element.classList.add("active");
element.classList.remove("inactive");
element.classList.toggle("visible");
element.classList.contains("active"); // true
element.classList.replace("old", "new");
```

## Style Modification

```javascript
element.style.color = "red";
element.style.backgroundColor = "blue";
element.style.cssText = "color: red; font-size: 16px;";
```

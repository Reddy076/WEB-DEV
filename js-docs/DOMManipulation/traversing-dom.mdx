# Traversing the DOM

Navigating between DOM nodes using parent, child, and sibling relationships.

## Parent Navigation

```javascript
element.parentNode;              // Parent (any node type)
element.parentElement;           // Parent (element only)
element.closest(".container");   // Nearest ancestor matching selector
```

## Child Navigation

```javascript
element.childNodes;              // All children (includes text nodes)
element.children;                // Element children only
element.firstChild;              // First child (any node)
element.firstElementChild;       // First element child
element.lastChild;               // Last child (any node)
element.lastElementChild;        // Last element child
```

## Sibling Navigation

```javascript
element.nextSibling;             // Next (any node)
element.nextElementSibling;      // Next element
element.previousSibling;         // Previous (any node)
element.previousElementSibling;  // Previous element
```

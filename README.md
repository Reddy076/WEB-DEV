# CSS Master Docs
A comprehensive, interactive CSS documentation site built with React and Docusaurus.

## Features
- **Interactive Playgrounds**: Live coding for HTML/CSS examples.
- **Detailed Topics**: From Syntax to Flexbox/Grid.
- **Mobile First**: Responsive design.
- **Interview Prep**: Questions included in every section.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm start
   ```
   Opens [http://localhost:3000](http://localhost:3000).

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Project Structure
- `/docs`: MDX content files.
- `/src/components/Playground`: Interactive code editor component.
- `/src/css`: Custom styles.
- `docusaurus.config.js`: Site configuration.

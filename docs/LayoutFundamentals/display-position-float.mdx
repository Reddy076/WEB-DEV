# Display, Position, Float

Tools for breaking the normal flow.

## 1. Position Property

- **Static**: Default.
- **Relative**: Offset from original spot. Space remains reserved.
- **Absolute**: Removed from flow. Relative to nearest *positioned* parent.
- **Fixed**: Relative to Viewport (Screen). Sticky navbars.
- **Sticky**: Hybrid. Relative until scrolled passed, then Fixed.

## 2. Float
`float: left` / `right`.
Originally used to wrap text around images.
Abused for grid layouts until 2010. **Do not use for layout anymore.**

## 3. Z-Index
Controls stacking order.
- **Rules**: Works only on positioned elements (`relative/absolute/fixed`).
- Elements with higher Z index appear on top.

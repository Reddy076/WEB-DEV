# Layout with CSS

Understanding **Layout** is distinguishing between the "Normal Flow" and the special layout modes (Flex/Grid).

## 1. Normal Flow
Unless you specify otherwise (`float`, `position`, `flex`), browsers place elements in Normal Flow.
- **Block**: Vertical stack.
- **Inline**: Human text flow (left to right, wrapping).

## 2. Block vs Inline vs Inline-Block

| Mode | Line | Width/Height | Margins |
| :--- | :--- | :--- | :--- |
| **Block** | New Line | Respected | Respected |
| **Inline** | Same Line | Ignored | Horizontal only |
| **Inline-Block**| Same Line | Respected | Respected |

## 3. Centering Reference

- **Horizontal Text**: `text-align: center`.
- **Horizontal Block**: `margin: 0 auto` (needs width).
- **Vertical**: `display: flex; align-items: center;`.

# Media Queries

Media queries allow you to modify your site depending on the device characteristics (width, height, orientation). This is the key to **Responsiveness**.

## 1. Syntax

```css
@media (condition) {
  /* Rules to apply if condition is true */
}
```

## 2. Conditions

## 2. Conditions & Strategies

### Mobile First (Recommended)
We style for mobile (default), then add rules for larger screens. Use `min-width`.
```css
/* Base styles (Mobile) */
body { font-size: 14px; }

/* Tablet */
@media (min-width: 768px) {
  body { font-size: 16px; }
}

/* Desktop */
@media (min-width: 1024px) {
  body { font-size: 18px; }
}
```

### Desktop First (Legacy)
We style for desktop, then fix issues for smaller screens. Use `max-width`.
```css
/* Base (Desktop) */
body { font-size: 18px; }

@media (max-width: 768px) {
  /* Fix for mobile */
  body { font-size: 14px; }
}
```

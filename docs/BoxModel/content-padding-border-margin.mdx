# Content, Padding, Border, Margin

A detailed breakdown of the properties that control the Box Model.

## 1. Margin (Outer Spacing)
Creates space **between** elements.
- **Shorthand**: `margin: 10px 20px 10px 20px` (Top Right Bottom Left - Clockwise).
- **Auto**: `margin: 0 auto` centers a block horizontally.
- **Negative**: `margin-top: -10px` pulls element up.
- **Collapse**: Adjacent vertical margins merge to the largest one.

## 2. Padding (Inner Spacing)
Creates space **inside** the element.
- **Shorthand**: Same as margin (Clockwise).
- **No Negative**: Cannot have negative padding.
- Used to give content "breathing room".

## 3. Border (The Line)
- `border-width`: `1px`.
- `border-style`: `solid`, `dashed`, `dotted`, `none`.
- `border-color`: `red`.
- **Shorthand**: `border: 1px solid red;`

## 4. Content
Controlled by `width` and `height`.
- `min-width` / `max-width`: Limits.
- `min-height` / `max-height`.

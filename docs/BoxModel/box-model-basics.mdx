# Box Model Basics

The **Box Model** is the core concept of CSS layout. It implies that every element is a rectangular box.

## 1. Composition
The box consists of 4 layers (Inside -> Outside):

```text
+---------------------------+
|        Margin             |
|  +---------------------+  |
|  |      Border         |  |
|  |  +---------------+  |  |
|  |  |   Padding     |  |  |
|  |  |  +---------+  |  |  |
|  |  |  | Content |  |  |  |
|  |  |  +---------+  |  |  |
|  |  +---------------+  |  |
|  +---------------------+  |
+---------------------------+
```

1.  **Content**: The actual content (text, images).
2.  **Padding**: Space between content and border.
3.  **Border**: The edge of the element.
4.  **Margin**: Space outside the border.

## 2. Default Behavior (`content-box`)
By default, if you set `width: 100px`, that is the width of the **Content**.
If you add `padding: 20px`, the total width becomes `100 + 20 + 20 = 140px`.
*This is often confusing for beginners.*

## 3. Modern Behavior (`border-box`)
We can change this using `box-sizing: border-box`.
If you set `width: 100px`, that is the **Total** width.
If you add `padding: 20px`, the content shrinks to `60px`, but the box stays `100px`.

**Universal Reset:**
```css
* {
  box-sizing: border-box;
}
```

## 5. Important Interview Questions

<details>
  <summary>**1. Explain the Box Model.**</summary>
  It's the specification that determines how elements are sized and rendered. Content > Padding > Border > Margin.
</details>

<details>
  <summary>**2. Why use `box-sizing: border-box`?**</summary>
  It makes layout math predictable. `width: 50%` means half the screen, regardless of borders or padding used.
</details>

<details>
  <summary>**3. Which parts of the box are transparent?**</summary>
  Content (usually), Padding (always matches background), and Margin (Always transparent).
</details>

<details>
  <summary>**4. Does `background-color` cover the margin?**</summary>
  No. It covers Content, Padding, and Border (if border is transparent/dotted). Margin remains transparent.
</details>

<details>
  <summary>**5. Calculating width (content-box): 100px width, 10px padding, 5px border?**</summary>
  100 + 20 (padding L/R) + 10 (border L/R) = **130px**.
</details>

# Styling Text and Fonts

Text is the primary interface of the web. This section covers formatting text appearance.

## 1. Text Color
- `color`: Sets the foreground color of the text.

## 2. Text Alignment
- `text-align`: `left`, `right`, `center`, `justify`.
- *Note*: `justify` adds varying space between words to ensure lines end at the same point.

## 3. Text Decoration
Used for lines.
- `text-decoration: underline`: Standard link style.
- `text-decoration: none`: Removes underlines.
- `text-decoration: line-through`: Deleted text style.

## 4. Text Transformation
- `uppercase`: ALL CAPS.
- `lowercase`: no caps.
- `capitalize`: First Letter Of Each Word.

## 5. Spacing
- `letter-spacing`: Space between characters.
- `word-spacing`: Space between words.
- `line-height`: Vertical space. **Golden rule**: Use unitless numbers (e.g., `1.5` means 1.5x the font size).

## 6. Shadows
`text-shadow: offset-x offset-y blur color;`
```css
h1 {
  text-shadow: 2px 2px 5px rgba(0,0,0,0.5);
}
```

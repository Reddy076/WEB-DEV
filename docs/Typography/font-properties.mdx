# Font Properties

While the previous section styling text layout (alignment, decoration), this section focuses specifically on the **typeface** itself.

## 1. Font Family

Defines the font to be used.
```css
font-family: "Open Sans", Helvetica, Arial, sans-serif;
```
- **Fallback System**: If "Open Sans" isn't found, it tries Helvetica, then Arial.
- **Generic Family**: Always end with one (`serif`, `sans-serif`, `monospace`, `cursive`).

## 2. Web Fonts
You are not limited to system fonts (`Arial`, `Times`).
- **Google Fonts**: `<link href="...">`
- **@font-face**: Load local files (`.woff2`).

## 3. Font Size
- **px**: Absolute. `16px`.
- **em**: Relative to parent. `1.2em` = 1.2 x Parent Size.
- **rem**: Relative to Root (HTML). **Best for accessibility**.
- **vw**: Viewport Width.

## 4. Font Weight
Thickness of characters.
- Keywords: `normal` (400), `bold` (700).
- Numeric: `100` (Thin) -> `900` (Black).
- *Note: The font file must support the specific weight.*

## 5. Font Style
- `italic`: Real italic glyphs.
- `oblique`: Slanted normal glyphs.

## 6. Font Variant
- `small-caps`: LOWERCASE LETTERS BECOME SMALL CAPITALS.

## 8. Important Interview Questions

<details>
  <summary>**1. Why use `rem` instead of `px`?**</summary>
  `rem` respects the user's browser font size settings (default 16px, but visually impaired users might set it to 24px). If you hardcode `16px`, you override their preference.
</details>

<details>
  <summary>**2. What is a "Web Safe Font"?**</summary>
  Fonts that are pre-installed on almost all devices (Arial, Times New Roman, Courier). You don't need to download them.
</details>

<details>
  <summary>**3. Explain `@font-face`.**</summary>
  A CSS rule that allows you to load custom fonts from a server.
  ```css
  @font-face {
    font-family: 'MyFont';
    src: url('myfont.woff2');
  }
  ```
</details>

<details>
  <summary>**4. `font-display: swap`?**</summary>
  Performance property. It tells the browser to display text in a fallback font immediately, then "swap" to the custom font once it loads. Prevents invisible text.
</details>

<details>
  <summary>**5. What is a Variable Font?**</summary>
  A newer font technology where one single file contains all weights and styles (thin, bold, italic), adjustable via `font-variation-settings`.
</details>

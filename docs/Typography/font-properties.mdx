# Font Properties

While the previous section styling text layout (alignment, decoration), this section focuses specifically on the **typeface** itself.

## 1. Font Family

Defines the font to be used.
```css
font-family: "Open Sans", Helvetica, Arial, sans-serif;
```
- **Fallback System**: If "Open Sans" isn't found, it tries Helvetica, then Arial.
- **Generic Family**: Always end with one (`serif`, `sans-serif`, `monospace`, `cursive`).

## 2. Web Fonts
You are not limited to system fonts (`Arial`, `Times`).
- **Google Fonts**: `<link href="...">`
- **@font-face**: Load local files (`.woff2`).

## 3. Font Size
- **px**: Absolute. `16px`.
- **em**: Relative to parent. `1.2em` = 1.2 x Parent Size.
- **rem**: Relative to Root (HTML). **Best for accessibility**.
- **vw**: Viewport Width.

## 4. Font Weight
Thickness of characters.
- Keywords: `normal` (400), `bold` (700).
- Numeric: `100` (Thin) -> `900` (Black).
- *Note: The font file must support the specific weight.*

## 5. Font Style
- `italic`: Real italic glyphs.
- `oblique`: Slanted normal glyphs.

## 6. Font Variant
- `small-caps`: LOWERCASE LETTERS BECOME SMALL CAPITALS.

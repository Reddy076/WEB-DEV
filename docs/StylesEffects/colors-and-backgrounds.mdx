# Colors and Backgrounds

## 1. Color Formats
- **Keywords**: `red`, `blue`, `cornflowerblue`.
- **Hex**: `#FF0000` (Red), `#00FF00` (Green).
- **RGB**: `rgb(255, 0, 0)`.
- **RGBA**: `rgba(255, 0, 0, 0.5)` (0.5 = 50% opacity).
- **HSL**: `hsl(Hue, Sat%, Light%)`. 
    - Hue: 0-360 wheel.
    - Sat: 0% (Gray) - 100% (Vibrant).
    - Light: 0% (Black) - 50% (Pure) - 100% (White).

## 2. Background Properties
- `background-color`: Fills the box.
- `background-image`: `url('img.jpg')`.
- `background-repeat`: `no-repeat` / `repeat`.
- `background-size`: 
    - `cover` (Fills box, crops).
    - `contain` (Matches max dimension, letterboxes).
- `background-position`: `center center`.

## 3. Gradients
Gradients are images generated by the browser.
- **Linear**: `linear-gradient(direction, color1, color2)`.
- **Radial**: `radial-gradient(shape, color1, color2)`.

## 5. Important Interview Questions

<details>
  <summary>**1. `display: none` vs `opacity: 0` vs `visibility: hidden`?**</summary>
  - `display: none`: Removed from layout. No space.
  - `opacity: 0`: Invisible, takes space, **interactive** (clickable).
  - `visibility: hidden`: Invisible, takes space, **not interactive**.
</details>

<details>
  <summary>**2. What is `background-attachment: fixed`?**</summary>
  Parallax effect. The background image stays stuck to the viewport while the page content scrolls.
</details>

<details>
  <summary>**3. Difference between `opacity` and `rgba` alpha?**</summary>
  `opacity` property fades the element *and children*. `rgba` background only fades the background color; children stay solid.
</details>

<details>
  <summary>**4. How to use a sprite sheet?**</summary>
  One large image with many icons. Use `background-image` + `background-position` to show only the slice you want through a fixed width/height container. Saves HTTP requests.
</details>

<details>
  <summary>**5. CSS variables?**</summary>
  `--primary: #ff0000;` then `color: var(--primary);`.
</details>

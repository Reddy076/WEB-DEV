# Colors and Backgrounds

## 1. Color Formats
- **Keywords**: `red`, `blue`, `cornflowerblue`.
- **Hex**: `#FF0000` (Red), `#00FF00` (Green).
- **RGB**: `rgb(255, 0, 0)`.
- **RGBA**: `rgba(255, 0, 0, 0.5)` (0.5 = 50% opacity).
- **HSL**: `hsl(Hue, Sat%, Light%)`. 
    - Hue: 0-360 wheel.
    - Sat: 0% (Gray) - 100% (Vibrant).
    - Light: 0% (Black) - 50% (Pure) - 100% (White).

## 2. Background Properties
- `background-color`: Fills the box.
- `background-image`: `url('img.jpg')`.
- `background-repeat`: `no-repeat` / `repeat`.
- `background-size`: 
    - `cover` (Fills box, crops).
    - `contain` (Matches max dimension, letterboxes).
- `background-position`: `center center`.

## 3. Gradients
Gradients are images generated by the browser.
- **Linear**: `linear-gradient(direction, color1, color2)`.
- **Radial**: `radial-gradient(shape, color1, color2)`.

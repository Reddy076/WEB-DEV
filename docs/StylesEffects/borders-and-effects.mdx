# Borders and Effects

## 1. Border Radius
Softens the look of the web.
- `border-radius: 5px`: Small round.
- `border-radius: 50%`: Circular.

## 2. Box Shadow
Adds depth.
- `box-shadow: 10px 10px 5px grey;`
- Order: X-offset, Y-offset, Blur, Color.
- `inset`: Shadow inside the box.

## 3. Outline
Similar to border, but:
- Does not take up space (overlaps).
- Cannot be circular (in older browsers).
- Often used for Accessibility focus rings.

## 4. Filters
Photoshop-like effects in the browser.
- `blur(5px)`
- `grayscale(100%)`
- `brightness(150%)`
- `drop-shadow()` (Better than box-shadow for PNGs).

## 6. Important Interview Questions

<details>
  <summary>**1. `box-shadow` vs `border` layout impact?**</summary>
  Border takes up space (adds to width). Box-shadow does not; it acts like paint spilling out.
</details>

<details>
  <summary>**2. How to create a donut shape?**</summary>
  Circle with a thick, transparent border? Actually better: Circle with a `border` colored, but strictly it's usually `border-radius: 50%` + `border: 20px solid red` + transparent background.
</details>

<details>
  <summary>**3. What is `clip-path`?**</summary>
  Allows cutting the element into shapes (triangle, polygon) like using scissors.
</details>

<details>
  <summary>**4. Default outline style?**</summary>
  Browsers add a blue/black outline on `:focus`. Never remove it (`outline: none`) without replacing it (e.g., with a `box-shadow`), or you kill accessibility.
</details>

# Transitions & Animations

Bringing the web to life.

## 1. Transitions
Smoothly change a property from State A to State B (e.g., Hover).
**Syntax**: `transition: property duration timing-function delay;`

```css
button {
  background: blue;
  transition: background 0.3s ease-in-out;
}
button:hover {
  background: red;
}
```

- **Timing Functions**: `linear`, `ease`, `ease-in`, `ease-out`, `cubic-bezier`.

## 2. Animations (@keyframes)
Complex multi-stage animations.

```css
@keyframes slideIn {
  0% { transform: translateX(-100%); }
  100% { transform: translateX(0); }
}

.box {
  animation: slideIn 1s forwards;
}
```

### Properties
- `animation-name`: Keyframe name.
- `animation-duration`: `2s`.
- `animation-iteration-count`: `infinite` or number.
- `animation-direction`: `alternate` (back and forth).

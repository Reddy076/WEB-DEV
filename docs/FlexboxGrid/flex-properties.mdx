# Flex Properties: Deep Dive

## 1. Container Properties

### `flex-direction`
- `row` (default): Left to right.
- `row-reverse`: Right to left.
- `column`: Top to bottom.
- `column-reverse`: Bottom to top.

### `justify-content` (Main Axis Alignment)
- `flex-start`: Pack to start.
- `flex-end`: Pack to end.
- `center`: Center pack.
- `space-between`: First at start, last at end, equal space inside.
- `space-around`: Equal space around each item.
- `space-evenly`: Equal space everywhere.

### `align-items` (Cross Axis Alignment)
- `stretch` (default): Stretch to fill container height.
- `flex-start` / `flex-end` / `center`.
- `baseline`: Align text baselines.

### `flex-wrap`
- `nowrap` (default): Shrink items to fit one line.
- `wrap`: Allow items to drop to next line.

## 2. Item Properties

### `flex-grow`
Unitless number. Percentage of *extra* space this item should take.
`flex-grow: 1` means "take up all remaining space".

### `flex-shrink`
Factor of shrinkability if space is tight.

### `flex-basis`
Initial size before growing/shrinking.

### Shorthand: `flex`
`flex: grow shrink basis;`
*Example*: `flex: 1 1 auto;`

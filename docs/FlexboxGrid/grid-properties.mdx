# Grid Properties

CSS Grid Layout (Grid) works on 2 Dimensions (Rows & Columns) simultaneously.

## 1. Container Properties

### `grid-template-columns` / `rows`
Defines the tracks.
- `100px 200px`: First col 100px, second 200px.
- `1fr 1fr`: Two equal columns (fractions of free space).
- `repeat(3, 1fr)`: Three equal columns.

### `gap`
Space between cells.
- `gap: 20px;`

### `grid-template-areas`
Semantic layout map.
```css
grid-template-areas:
  "header header"
  "sidebar content"
  "footer footer";
```

## 2. Item Properties

- `grid-column`: `start / end`.
    - `grid-column: 1 / 3` (Spans from line 1 to line 3).
    - `grid-column: span 2`.
- `grid-area`: Assign to name (`grid-area: header`).

## 4. Important Interview Questions

<details>
  <summary>**1. What is the `fr` unit?**</summary>
  Fraction of available space. It calculates *after* fixed widths (px) are subtracted.
</details>

<details>
  <summary>**2. `max-content` vs `min-content`?**</summary>
  - `min-content`: Narrowest possible (wrap text at longest word).
  - `max-content`: Widest possible (no wrapping).
</details>

<details>
  <summary>**3. Difference between Grid and Flexbox?**</summary>
  Grid locks items to the grid lines (X and Y). Flexbox flows items based on content size (X).
</details>

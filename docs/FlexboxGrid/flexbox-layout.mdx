# Flexbox Layout

**Flexbox** (Flexible Box Module) is designed for 1-dimensional layouts (Row or Column). It distributes space between items and aligns them.

## 1. The Container (`display: flex`)
Activating flexbox transforms the direct children into "flex items".

```css
.container {
  display: flex;
}
```

## 2. Flex Direction
Two axes:
- **Main Axis**: Default Horizontal (Row).
- **Cross Axis**: Default Vertical (Column).

We can potential swap these using `flex-direction: column`.

## 3. Key Benefits
- Vertical Centering!
- Equal height columns.
- Reordering items without changing HTML.

## 5. Important Interview Questions

<details>
  <summary>**1. Flexbox vs Grid?**</summary>
  - Flexbox: 1D (Line). Content-first.
  - Grid: 2D (Rows & Columns). Layout-first.
</details>

<details>
  <summary>**2. What is `justify-content`?**</summary>
  Aligns items along the **Main Axis**.
</details>

<details>
  <summary>**3. What is `align-items`?**</summary>
  Aligns items along the **Cross Axis**.
</details>

# Basic Forms

Forms are used to collect user input. The `<form>` element wraps various input elements.

### Interactive Form Example

Try interacting with the form below. Notice how different input types behave.

## Form Structure

```html
<form action="/submit-url" method="POST">
  <!-- Inputs go here -->
</form>
```

- **action**: Where to send the form data.
- **method**: HTTP method (GET or POST).

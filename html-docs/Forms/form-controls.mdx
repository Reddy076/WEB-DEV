# Form Controls

Beyond standard input types, HTML offers other controls for richer user interaction.

## Textarea
The `<textarea>` element defines a multi-line text input control.

```html
<textarea rows="4" cols="50">
Default text...
</textarea>
```

## Select & Option (Dropdown)
The `<select>` element creates a drop-down list.

```html
<select name="cars" id="cars">
  <option value="volvo">Volvo</option>
  <option value="saab">Saab</option>
  <option value="mercedes">Mercedes</option>
  <option value="audi">Audi</option>
</select>
```

### Attributes
- `multiple`: Allow multiple selections.
- `size`: Number of visible options.

### Grouping Options
Use `<optgroup>` to group related options.

```html
<select>
  <optgroup label="Swedish Cars">
    <option value="volvo">Volvo</option>
    <option value="saab">Saab</option>
  </optgroup>
</select>
```

## Buttons
The `<button>` element defines a clickable button.

```html
<button type="button" onclick="alert('Hello')">Click Me</button>
```

## Datalist
The `<datalist>` element specifies a list of pre-defined options for an `<input>` element. Users can still type their own value.

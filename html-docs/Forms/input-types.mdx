# Input Types

The `<input>` element is the most important form element. Its behavior depends heavily on the `type` attribute.

## Common Input Types

1.  **text**: Default single-line text field.
2.  **password**: Masked characters.
3.  **submit**: Button to submit form.
4.  **reset**: Button to reset form.
5.  **radio**: Select ONE from many.
6.  **checkbox**: Select ANY from many.
7.  **button**: Clickable button.

## HTML5 Input Types
These types provide better semantic meaning and mobile keyboard integration:

- `color`: Color picker.
- `date` / `time` / `datetime-local`: Native date/time pickers.
- `email`: Validates email syntax.
- `number`: Numeric input with spinners.
- `range`: Slider control.
- `search`: Styled for search queries.
- `tel`: Optimized keyboard for phone numbers.
- `url`: Validates URL syntax.

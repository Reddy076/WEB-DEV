# Links & Images

## Hyperlinks
HTML links are hyperlinks. You can click on a link and jump to another document.

```html
<a href="url">Link text</a>
```

### Attributes
- `href`: The destination URL.
- `target`: Where to open the link.
    - `_self`: Default. Opens in same window.
    - `_blank`: Opens in a new window/tab.
- `title`: Tooltip text on hover.

## Images
HTML images are defined with the `<img>` tag. It is empty, containing attributes only, and does not have a closing tag.

```html
<img src="url" alt="text">
```

### Attributes
- `src`: The path to the image.
- `alt`: Alternate text for screen readers or if image fails to load.
- `width`/`height`: Size in pixels.
- `loading`: `lazy` (defer loading) or `eager`.

## File Paths
- **Absolute URL**: Full web address (`https://www.google.com`)
- **Relative URL**: Path relative to current page (`images/logo.png`)
